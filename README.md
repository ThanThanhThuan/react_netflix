# React + Vite Netflix Clone
Streaming Service Clone (Netflix Style)

Description: A movie discovery application interacting with third-party APIs to provide browsing, searching, and video playback capabilities.

    Tech Stack: React, Axios, TMDB API, React Player, Intersection Observer.

    Key Features:

        Advanced API Integration: Configured Axios instances to fetch data from The Movie Database (TMDB) across multiple categories (Trending, Top Rated, Genres).

        Infinite Scrolling: Implemented the Intersection Observer API to detect when a user reaches the bottom of search results and automatically fetch the next page of data.

        Dynamic Routing: utilized URL parameters (/movie/:id) to fetch and display specific movie details and trailers.

        Video Embedding: Integrated react-player to safely embed YouTube trailers based on API results.
